# LLM

Configuración de modelos, prompts base y plantillas reutilizables.

## Sugerencia de estructura

```
llm/
├── models.yaml         # IDs de modelos y parámetros por defecto
├── prompts/            # prompts del sistema reutilizables
│   └── base.md
└── templates/          # plantillas con placeholders
```
