# Skills

Cada skill vive en su propia carpeta con un `SKILL.md`.

## Formato mínimo

```
skills/
└── mi-skill/
    ├── SKILL.md        # frontmatter (name, description) + instrucciones
    ├── scripts/        # opcional: helpers ejecutables
    └── resources/      # opcional: archivos de apoyo
```

`SKILL.md` debe tener frontmatter:

```yaml
---
name: mi-skill
description: Qué hace y cuándo dispararse.
---
```
